How I Created the Matrix Effect for My Portfolio

Published on June 29, 2025 • 5 min read

In this article, I’ll walk you through the entire process of creating this animation, using only HTML, CSS, and pure JavaScript. We’ll explore the basics of HTML Canvas to bring a dynamic and impactful background to life.

Step 1: Organizing Files and Folders

First, create a folder where you can create and import files in an organized way. Once you have done this, create three files with the same name (You can change it if you prefer, but I recommend that you keep the same name). index.html, style.css e script.js

Step 2: The HTML Structure

On your file index.html create a basic structure of an HTM file, now look for a section called <body></body> and inside it, copy and paste this script: <canvas id="matrix-canvas"></canvas>

After adding the canvas, don't forget to link your CSS and JavaScript files in your HTML. Add<link rel="stylesheet" href="style.css"> inside of <head> and <script src="script.js" defer></script> before closing the </body>

Step 3: The CSS Structure

For the effect to work correctly, we need to ensure that the canvas behaves like a background, in your style.css file copy and paste this script: 


#matrix-canvas {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    z-index: 0; /* Ensures the canvas stays behind the contentS */
                    
}

Step 4: Logic with JavaScript

Moving on to the cherry on the cake, which would be the .js file, this is where the magic happens. We will use all the functions so that your browser can create the effect of falling letters.

In the code below, I used the classic green and black colors. In my portfolio, I just changed the values ​​of fillStyleto match the website's purple visual identity. With your script.js file open, copy and paste this function structure: 


function iniciarEfeitoMatrix() {
    const canvas = document.getElementById('matrix-canvas');
    if (!canvas) return;

    const ctx = canvas.getContext('2d');
    canvas.width = window.innerWidth;
    canvas.height = window.innerHeight;

    const alphabet = 'アァカサタナハマヤャラワガザダバパイィキシチニヒミリヰギジヂビピウゥクスツヌフムユュルグズブプエェケセテネヘメレヱゲゼデベペオォコソトノホモヨョロヲゴゾドボポヴッンABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789';
    const fontSize = 16;
    const columns = Math.floor(canvas.width / fontSize);
    const rainDrops = Array.from({ length: columns }).fill(1);

    const desenhar = () => {
        ctx.fillStyle = 'rgba(5, 29, 2, 0.01)';
        ctx.fillRect(0, 0, canvas.width, canvas.height);
        ctx.fillStyle = '#00FF00';
        ctx.font = fontSize + 'px monospace';
            for (let i = 0; i < rainDrops.length; i++) {
                const text = alphabet.charAt(Math.floor(Math.random() * alphabet.length));
                ctx.fillText(text, i * fontSize, rainDrops[i] * fontSize);
            if (rainDrops[i] * fontSize > canvas.height && Math.random() > 0.975) {
                rainDrops[i] = 0;
            }
            rainDrops[i]++;
        }
    };

    setInterval(desenhar, 33);

    window.addEventListener('resize', () => {
    canvas.width = window.innerWidth;
        canvas.height = window.innerHeight;
        // Recalcula o número de colunas, mas não precisa resetar as gotas
    });
}  

document.addEventListener('DOMContentLoaded', () => {
    iniciarEfeitoMatrix();
});

Bonus: Customizing the Animation

I will teach you how to change the characters that appear and this change is actually super simple, in your file find the following function: const alphabet.
It is responsible for sending the order of the characters, in this field you can put any character as long as the browser supports it and even some types of emojis, if you want to test it, I will leave an example of how you can do it.

Ex: ABCDEFGHIJKLMNOPQRSTUVWXYZ123456789

Warning: When replacing any value within the fieldconst alphabet = '', don't forget to keep the single quotes, otherwise the code won't work.

And there you have it! With a simple HTML structure, a little CSS, and the magic of JavaScript Canvas, we’ve created an iconic, fully functional background animation.
